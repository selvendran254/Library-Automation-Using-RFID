How to Connect Real RFID

Option 1 — HID Reader (Easy):

  1. Plug USB RFID reader
  2. Set mode = HID
  3. Open Issue/Return page
  4. Focus input field & scan tag

Option 2 — Serial Reader:

  1. Plug USB serial RFID reader
  2. Find port: ls /dev/ttyUSB*
  3. Set mode = Serial, select port
  4. Click Start Reader

Tag format must match database: BOOK-001, MEM-CARD-001, etc.
