# Library Automation Using RFID

This project automates book issue/return using RFID tags and an MFRC522 RFID reader with Arduino.
Features include book scanning, user authentication, serial logging, and expandable database support.

## Features
- Scan RFID tag of books and users
- Real-time serial logging of issue/return events
- Store book/user details in CSV or external DB
- LCD-based status display
- Buzzer alert for invalid tag scan

## Hardware
- Arduino Uno / Nano
- MFRC522 RFID Module
- LCD 16x2 (I2C recommended)
- Buzzer + LED indicators
- RFID tags/cards
